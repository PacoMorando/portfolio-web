Github repository Play store release

A pesar de que en la play store ya existen otras aplicaciones que emulan una regla con valores reales, las que existen parecen estar diseñadas para usuarios generales y para usos genericos.
Es por esto que me decidí por diseñar y desarrollar una herramienta más enfocada al uso que le damos los arquitectos y diseñadores a este tipo de herramientas.

Las claves para enfocar el diseño fueron:

  • Un set de escalas de uso común fácil de configurar.
  • Una calculadora para convertir valores.
  • Escalas gráficas como referiencia visual.
  • Interfaz sencilla y de facil manejo.

El desarrollo de esta aplicación es un ejercicio que realicé mientras estudiaba sobre arquitectura del software MV*, razón principal por la que el código podría parecer "sobre diseñado" o excesivamente modularizado. Lo que buscaba era poner en práctica los conceptos de diseño de software, así como el reparto de responsabilidades y la interacción entre los paquetes que conforman una arquitectura MVC.
